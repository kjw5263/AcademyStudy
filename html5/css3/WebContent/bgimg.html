배경 이미지의 속성들

  1. background-repeat: ; 반복 지정
    • repeat
    • repeat-x
    • repeat-y
    • no-repeat
  2. background-size : ; 크기 지정
    • contain : 이미지 하나를 온전히 보이면서 영역을 덮기
    • cover : 이미지 하나로 영역을 다 덮기(이미지 잘려나갈 수 있음)
    • Xn Xy
  3. 위치 지정
    • left/center/right top/center/bottom : 창 크기 변환시 주요로 보여줄 화면 부위
    • Xn Yn
  4. 스크롤 여부 지정
    • scroll
    • fixed
    • local























































































































background-attachment: ; 스크롤 여부 지정

 scroll은 브라우저 스크롤바에는 스크롤, 자신 영역의 스크롤 바에는 고정
 fixed는 브라우저 스크롤바와 자신 영역의 스크롤바에 모두 고정
 local 은 브라우저 스크롤바와 스크롤, 자신 영역의 스크롤바에 모드 스크롤
