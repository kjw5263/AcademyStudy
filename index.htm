html5에서 추가된 폼 요소들

------------------------------ 시간 관련 태그 ------------------------------

input type="time" 시간(시분)을 선택

input type="week" 주 단위 선택 (한주 통째로)

input type="month" 월 단위 선택 (월 통째로)

input type="date" 날짜를 선택

input type="datetime-local" 시간(시분)을 선택




------------------------------ 연락처 관련 태그 ------------------------------

input type="tel" 숫자 키패드

모바일 브라우저에서 가상 키패드가 숫자형식으로 바뀔 수 있고 전화걸기 가능

input type="email" 연락처(이메일 주소)를 지정

모바일 브라우저에서 가상 키패드가 영문 포함형식으로 바뀔 수 있고,
브라우저 자체에서 이메일 형식이 맞는지 검사해줌

input type="url" URL 입력받기(프로토콜 입력 필요)

모바일 브라우저에서 가상 키패드가 영문(https://등) 포함하는 형식으로 바뀔 수 있고,
브라우저에서 알맞은 웹주소 형식인지(프로토콜 여부)를 체크함

input type="number" 숫자 지정

숫자값을 포함하는 형식. 증감 버튼이 생성
최소값과 최대값을 속성으로 지정할 수 있고,
증감 간격을 속성으로 지정할 수 있다.

just number
min&max

input type="range" 범위 내의 값을 지정

input type="color" 컬러 값을 지정

html5에서 추가된 속성들

type=text autofocus -자동포커싱 지정(입력 대기 상태)


type=text autocomplete -자동완성 지원
