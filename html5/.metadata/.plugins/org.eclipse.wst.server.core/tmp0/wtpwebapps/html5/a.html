target 속성값

  • target="" 현재 위치에 열기
  • target="_blank" 새탭에서 열기
  • target="_parent" 한단계 상위창에 열기
  • target="_top" (프레임으로 나뉨없이)전체창에 열기

링크로 보여줄 수 있는 종류 - 브라우저가 처리 가능한 파일(종류)인 경우

  1. 외부 사이트(url) pixabay
  2. 외부 사이트2(url) pixels
  3. 웹페이지(*.html / *.htm) 이미지 수업보기
  4. 이미지(*.jpg / *.gif / *.png)
    5. drop.jpg arrow-up.png arrow.gif
  5. 동영상(*.ogg / *.mp4 / *.webm) 나무
  6. 텍스트파일(*.txt)
    7. 이클립스 다운받기, zip 파일은 다운로드 됩니다

링크로 보여줄 수 없는 종류 -브라우저가 처리 불가능한 파일(종류)인 경우 다운됨

img.zip
zip, hwp등의 파일은 다운로드 됨

 </a>
